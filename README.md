# Still Unnamed

A small mobile-first web app for a four-piece band — set list, notes per song per member, and tracks (tabs / YouTube / recordings). Single-page HTML/CSS/JS frontend, Google Apps Script + Google Sheet backend.

Hosted on GitHub Pages. Add to home screen on a phone for a fullscreen app feel.

## Files

- `index.html` — the app (production, wired to the Apps Script backend)
- `manifest.webmanifest` — PWA manifest
- `icon-*.png`, `apple-touch-icon.png`, `favicon-32.png` — app icons
- `mockup.html` — visual design reference, not deployed
- `apps-script/Code.gs` — Apps Script backend, deployed as a Google web app
- `apps-script/SETUP.md` — backend setup walkthrough
- `PLAN.md` — design/decisions doc
- `build-icon.py` — regenerates the app icons from scratch (Pillow)
